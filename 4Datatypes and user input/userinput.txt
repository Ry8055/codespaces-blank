USER INPUT 
To ask for the input from the user , default datatype is string . 
name = input("enter your name here")
print(name)

output : 
Entered name by the user . 

if want to take input in numerical that means age or any number that we can do .

print(age)
note : if user give string input instead of number , it will through error .
